Совет. Гугление дополнительной информации производить по ключевой фразе "gnu screen".

## Единственный сеанс GNU Screen

| Команда | Описание |
|---|---|
| `screen -ls` | вывести список всех Screen-сеансов. |
| `screen` | запустить новый безымянный Screen-сеанс. |
| `screen -r` | возобновить отключённый Screen-сеанс. |
| `screen -d` | отключиться от текущего Screen-сеанса. |
| `screen -x` | подключиться к активному Screen-сеанс в режиме multiuser. |

## Множество сеансов GNU Screen
| Команда | Описание |
|---|---|
| `screen -ls` | вывести список всех Screen-сеансов. |
| `screen -S Atlas` | создать сеанс Screen с названием Atlas.|
| `screen -r A` или `screen -r At` и т.д. | возобновить сеанс Screen, указав часть его имени. |
| `screen -r 230919` | возобновить сеанс Screen, указав его полный номер. |
| `screen -d At` | отключиться от текущего Screen-сеанса, указав часть его имени. |
| `screen -d 2309` | отключиться от текущего Screen-сеанса, указав часть его номера. |
| `screen -x Atlas` | подключиться к активному Screen-сеанс в режиме multiuser, указав его полное имя. |
| `screen -x 23` | подключиться к активному Screen-сеанс в режиме multiuser, указав часть его номера. |

## Часто  используемые сочетания клавиш внутри GNU Screen
| Сочетание клавиш | Описание |
|---|---|
| `Ctrl+a` `d` | покинуть текущий сеанс screen.|
| `Ctrl+a` `s` | заморозка вывода stdout в активном окне. Ввод продолжает работать. Часто нажимается по ошибке. |
| `Ctrl+a` `q` | разморозка вывода stdout в активном окне. |
| `Ctrl+a` `x` | **заблокировать текущий сеанс Screen с выводом запроса пароля текущего пользователя.**  Часто нажимается по ошибке. Как назло, именно в этот момент пароль вылетает из головы! |
|---|
| `Ctrl+a` `c` | создать новое окно. |
| `Ctrl+a` `"` | вывести меню со всеми окнами в текущем сеансе screen'а с возможность перехода в выбранное окно. |
| `Ctrl+a` `1` | переключиться на окно номер "1" или вывести в текущем регионе окно номер "1". |
| `Ctrl+a` `n` | переключиться на следующее по порядку окно. |
| `Ctrl+a` `p` | переключиться на предыдущее по порядку окно. |
| --- |
| `Ctrl+a` `S` | разделить горизонтально активное окно/регион на два региона. |
| `Ctrl+a` `|` | разделить вертикально активное окно/регион на два региона. |
| `Ctrl+a` `Tab`  (или `Shift Tab`) | переключиться между видимыми регионами по часовой или против часовой стрелки. |
| `Ctrl+a` `:` `resize -10` | уменьшить размер текущего региона на десять строк или десять знакомест, в зависимости от схемы разделения окна. |
| `Ctrl+a` `:` `resize -v -10` | уменьшить размер текущего региона на десять строк.  |
| `Ctrl+a` `:` `resize -h +10` | увеличить размер текущего региона на десять знакомест. |
| `Ctrl+a` `:` `resize 10` | установить размер текущего региона в десять строк или десять знакомест, в зависимости от схемы разделения окна. |
| `Ctrl+a` `X` | закрыть активный регион. |
|---|---|
| `Ctrl+a` `[` или `Esc` | переход в режим копирования/вставки. Этот режим также используется для изучения вывода команды, когда интересующие строки скрылись за верхним краем экрана. |
| `↑` | переместить курсор вверх и прокрутить экран вниз. |
| `↓` | переместить крусор вниз и прокрутить экран вверх. |
| `q` или `Esc` | выход из режима копирования/вставки. |
|---|---|
