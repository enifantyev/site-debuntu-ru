Unix Programmer's Manual, Third Edition K. Thompson, D. M. Ritchie February, 1973

Начиная с седьмой редакции Unix (1979 год) приглашение оболочки sh для ввода команд для обычного пользователя, по умолчанию выглядит так:
$ _

А для суперюзера так:
# _

Обратите на это внимание при чтении материалов.
